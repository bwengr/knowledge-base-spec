# Changelog

All notable changes to this specification will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2026-05-02

### Added

- Initial specification
- JSON Schema for validation
- Discovery via `<link rel="knowledge-base">`, sitemap, and robots.txt
- Section types: blog-post, service-page, portfolio, faq, general
- Chunk structure with position ordering
- Examples: basic.json, bw-engineering.json
- Automation example with GitHub Actions workflow
